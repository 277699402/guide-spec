## 1 命名规则
### 1.1 目录命名
##### 全部采用小写方式，以下划线分隔。
例：scripts, styles, images, data_models
##### 项目目录说明
css:放置样式文件

fonts:放置字体文件

images:放置图片文件

js:放置js文件
