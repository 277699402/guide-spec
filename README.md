## 1 命名规则
### 1.1 目录命名
##### 全部采用小写方式，以下划线分隔。
例：css, fonts, images, js, data_models


### 1.2 其他文件命名
##### 参照目录命名规则。
html文件命名 例：error_report.html

js文件命名 例：account_model.js

css,less文件命名 例：retina_sprites.less
